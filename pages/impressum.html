Allgemeines zur Datenverarbeitung
Diese Website erhebt und speichert keine personenbezogenen Daten. Es werden keine Cookies gesetzt und keine Tracking- oder Analysetools eingesetzt.

Hosting
Diese Website wird über GitHub Pages gehostet. Beim Aufruf der Website werden durch den Hosting-Anbieter automatisch technische Daten erfasst (z. B. IP-Adresse, Zeitpunkt des Zugriffs, Browsertyp). Diese Daten werden zur Bereitstellung der Website benötigt und nach kurzer Zeit automatisch gelöscht. Weitere Informationen finden Sie in der Datenschutzerklärung von GitHub.

Eingebettete Inhalte
Diese Website kann eingebettete Videos enthalten. Eingebettete Videos werden erst nach Ihrer ausdrücklichen Zustimmung (Klick) geladen. Erst dann wird eine Verbindung zum jeweiligen Anbieter hergestellt und es können Daten übertragen werden. Ohne Ihre Interaktion werden keine Daten an Drittanbieter weitergegeben.

Externe Links
Diese Website enthält Links zu externen Websites (z. B. Instagram, Google Forms). Beim Anklicken dieser Links verlassen Sie unsere Website. Für die Datenverarbeitung durch die Betreiber dieser externen Seiten sind wir nicht verantwortlich.

SSL-Verschlüsselung
Diese Website nutzt aus Sicherheitsgründen eine SSL-Verschlüsselung (HTTPS).
